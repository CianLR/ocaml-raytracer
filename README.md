# ocaml-raytracer
A Raytracing library in OCaml

<img src="https://github.com/cianlr/ocaml-raytracer/raw/main/screenshot.png">

## Building
To build the raytracer example program run
```
dune build
```

## Running
To run the example run
```
dune exec src/main.exe
```
